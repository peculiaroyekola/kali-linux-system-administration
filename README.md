# Kali Linux System Administration Project

Practical exploration of Linux system administration and security concepts using Kali Linux.

---

## Project Overview

This project involved installing and configuring Kali Linux in a virtual machine environment and performing several administrative and security-related tasks.

---

## Tasks Performed

Installed Kali Linux in VirtualBox

Created and managed system users using GUI and command line

Installed and configured web services:

Apache  
PHP  
MariaDB

Deployed a WordPress website on localhost

Analyzed running processes using:

ps  
top  
htop

Configured file permissions using:

chmod  
chown
